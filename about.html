About Leonidas

der im ersten Semester Interface Design an der FH Potsdam studiert. Aus vorherigen Studien- und privaten Projekten ist Erfahrung im Programmieren mit Python sowie erste Erfahrungen mit HTML und CSS vorhanden. Mit dem Kurs "Visuelle Geschichten mit Code und Daten" wurde dem nicht nur mit dem Lernen von JavaScript Grundlagen ein weiterer Baustein hinzugefügt, sondern - und vor allem - durch das Arbeiten mit p5.js auf dem HTML-Canvas eine neue Perspektive auf das Arbeiten mit Code geschaffen.
Besonders inspiriert hat die Arbeit mit Perlin Noise als "Variation des Zufalls". Neben Perlin Noise ist die praktische Anwendung von p5.js für mich interessant, sodass durch geschickte Einbettung in "herkömmliche" Websites das volle Potential des Canvas in der Web- und Appentwicklung ausgeschöpft werden kann.
Ich hoffe, dass sich in naher Zukunft den existierenden Algorithmen noch ein paar weitere hinzufügen werden.
